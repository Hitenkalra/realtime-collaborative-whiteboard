# Real-Time Collaborative Interactive Whiteboard

A web-based collaborative whiteboard application that allows multiple users to draw and interact in real time. The project is designed for online collaboration, brainstorming, meetings, and learning environments.

## Features

* Real-time multi-user whiteboard interaction
* Interactive drawing interface
* Room-based collaboration system
* Low-latency synchronization
* Responsive web interface
* API-based communication
* Modern UI design

## Technology Stack

### Frontend

* HTML
* CSS
* JavaScript
* React.js

### Backend

* Node.js
* Express.js

### Real-Time Communication

* Socket.IO

## Project Objective

The main objective of this project is to create a collaborative platform where multiple users can interact and draw simultaneously with real-time synchronization.

## Installation

Clone the repository:

```bash
git clone https://github.com/Hitenkalra/realtime-collaborative-whiteboard.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

* Online teaching
* Team collaboration
* Brainstorming sessions
